Overview
This project presents a tomato leaf disease detection system built using EfficientNetB0 and TensorFlow. The model is trained on a real-field tomato dataset to classify multiple tomato plant diseases with optimized preprocessing, data augmentation, and balanced training techniques.
The project includes complete training, evaluation, visualization, and performance analysis pipelines for accurate agricultural disease classification.
Technologies Used
Python
TensorFlow / Keras
EfficientNetB0
NumPy
Scikit-learn
Matplotlib
Seaborn
Google Colab
Key Features
Tomato leaf disease classification
EfficientNetB0 transfer learning model
Real-field dataset training
Class balancing using computed class weights
Data augmentation for better generalization
Confusion matrix and performance visualization
Precision, Recall, F1-Score, and AUC evaluation
Bootstrap confidence interval analysis
Model saving and inference support
